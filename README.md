# RaspberryPi4-Niveaud-eau-Reservoir
Système de mesure du niveau d’eau avec Raspberry Pi

## Objectif du projet
- Surveiller le niveau d’eau dans un réservoir.
- Comparer la performance et la précision des systèmes numérique vs analogique.
- Fournir une solution pratique pour des applications domestiques ou industrielles.
- Alerter automatiquement par email si le niveau d’eau dépasse un seuil critique.
  
## Contenu
- Raspberry-NiveauEauNumerique-Reservoir.pdf – Documentation du système numérique.
- Raspberry-NiveauEauAnalogique-Reservoir.pdf – Documentation du système analogique.
- code_alert_email – Code Python pour la mesure du niveau et l’alerte par email.

## Technologies
- Raspberry Pi
- Capteur de niveau d’eau
- Langage Python
- Email pour alertes
